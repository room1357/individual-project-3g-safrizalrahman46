# pemrograman_mobile

A new Flutter project.

Flutter adalah sebuah framework open-source yang dikembangkan oleh Google untuk membangun aplikasi mobile dengan tampilan menarik dan performa tinggi. Flutter menggunakan bahasa pemrograman Dart dan konsep widget untuk membangun antarmuka pengguna yang responsif. Dengan fitur seperti hot reload, pengembangan aplikasi menjadi lebih cepat dan efisien.

Untuk proyek ini, saya Safrizal Rahman dari kelas SIB 3G mengembangkan aplikasi menggunakan Flutter yang mudah dipelajari dan diterapkan untuk pengembangan mobile modern.
